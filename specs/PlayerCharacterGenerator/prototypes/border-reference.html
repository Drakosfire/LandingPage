Best Practice: CONSISTENCY within the document.

  • Simple Solid Borders (what we use now) - Clean, modern, works well for dense data like character sheets
  • Monster Border Fancy - Official PHB stat block style, more decorative but heavier
  • Note Border - Good for callout boxes within a page

Avoid mixing multiple decorative borders on the same page. Pick ONE style and use it consistently.

Used for sidebar callouts and notes. Has a subtle torn/parchment edge effect. Good for "DM Tips" or "Designer Notes" boxes.

Used for flavor text and read-aloud boxes. Subtle decorative edge that doesn't overwhelm the italic text inside.
