## Extended Version (For Detailed Brief)

**Tool Name:** Player Character Generator (Character Generator)

**Core Function:** A web-based tool that helps players create D&D 5e characters through a guided 8-step wizard interface. Supports both manual step-by-step creation and AI-assisted generation. Produces PHB-style character sheets that can be exported as PDFs.

**Key Features:**
- Step-by-step wizard (ability scores, race, class, background, equipment, spells)
- AI-assisted character generation
- PHB-style 4-page character sheet rendering
- Portrait generation and upload
- Edit mode for post-creation adjustments
- Save/load functionality
- PDF export

**Visual Identity:**
- Part of the DungeonMind suite (alongside Statblock Generator, Card Generator, Rules Lawyer)
- D&D 5e official styling (PHB aesthetic)
- Professional tool for tabletop RPG players
- Combines traditional pen-and-paper feel with modern digital interface

**Icon Requirements:**
- Must work at multiple sizes (favicon 16x16, app icon 64x64, header icon 128x128)
- Should convey "character creation" and "D&D" at a glance
- Should feel approachable and helpful (guided wizard, not complex tool)
- Should match the aesthetic of other DungeonMind tools (professional, fantasy-themed, modern)

**Design Direction:**
- **Primary Concept:** Wizard hat + character sheet + dice
- **Secondary Concept:** Spellbook + quill + character sheet
- **Tertiary Concept:** Character portrait silhouette + creation tools

**Color Guidance:**
- Primary: Deep brown (#4a3728), parchment cream (#EEE5CE), gold accents (#C0AD6A)
- Secondary: D&D PHB red-brown text (#58180D), aged paper background
- Accent: Magical blue or purple for wizard/magic elements (optional)

**Style Notes:**
- Avoid overly complex details (won't scale well)
- Ensure high contrast for visibility at small sizes
- Consider both light and dark background variants
- Maintain consistency with other DungeonMind tool icons
